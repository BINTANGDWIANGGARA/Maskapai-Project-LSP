Maskapai Project LSP
Sistem pemesanan tiket pesawat berbasis web yang dibuat untuk kebutuhan Ujian Sertifikasi Kompetensi (LSP) – Junior Assistant Programmer.


Deskripsi

Project ini merupakan aplikasi web yang digunakan untuk melakukan pemesanan tiket pesawat secara online.
Pengguna dapat mencari tiket berdasarkan kota asal, tujuan, dan tanggal keberangkatan, kemudian memilih penerbangan dan melakukan proses pemesanan.

Aplikasi ini dibuat dengan alur sederhana namun tetap merepresentasikan sistem pemesanan tiket pada umumnya.

Fitur Utama
Login dan register pengguna
Pencarian tiket berdasarkan asal, tujuan, dan tanggal
Menampilkan daftar penerbangan
Pemilihan tiket dan jumlah penumpang
Menampilkan informasi harga
Proses booking tiket
Pengelolaan data (admin jika tersedia)
Teknologi yang Digunakan
Frontend: HTML, CSS, Bootstrap, JavaScript
Backend: PHP Native
Database: MySQL (phpMyAdmin)
Web Server: Apache (XAMPP)
Alur Sistem
Pengguna melakukan login atau register
Pengguna mencari tiket dengan mengisi data perjalanan
Sistem menampilkan daftar penerbangan yang tersedia
Pengguna memilih tiket
Pengguna melakukan pemesanan
Data tersimpan ke dalam database
